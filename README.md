# cloud-youtube (starter)

This repo contains a minimal starter for a cloud-native "YouTube-like" service.

Structure
- services/backend: minimal Node/Express API for local testing
- infra/: (placeholder) add Terraform/CloudFormation here
- services/frontend: (placeholder) add Next.js frontend

Next steps
1. Choose cloud provider (AWS / GCP / Azure).
2. Replace local object storage with S3 / Cloud Storage.
3. Implement resumable uploads and FFmpeg processing pipeline.
